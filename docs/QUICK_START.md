# SimHPC Free Tier — Quick Start

**What You Can Do**

* Run real simulations
* Change parameters and re-run
* Compare results
* Download PDF reports
* 10 Free runs per week

---

**Step-by-Step (Do This First)**

### 1. Run Your First Simulation

* Go to **Templates**
* Select any starter simulation
* Click **Run**

--> This creates your baseline

---

### 2. Wait for Completion

* Open **Job Queue**
* Status: *Queued → Running → Completed*

--> Free tier uses shared compute, so short waits are normal

---

### 3. View Results

* Open your completed job
* Look at:
  * Output graph
  * Runtime
  * Data

--> This is your reference result

---

### 4. Change ONE Parameter

Click **Modify / Re-run**

Change only one:
* Iterations (speed vs accuracy)
* Grid size (resolution)

--> Keep changes small

---

### 5. Re-run Simulation

* Click **Run** again
* Wait for completion

---

### 6. Compare Results

* Open both runs
* Compare:
  * Runtime
  * Output differences

--> This is where real value happens

---

### 7. Download Report

* Click **Download PDF**

--> You now have a shareable engineering artifact

---

**How to Get the Most Value**

Do this loop 3–5 times:
1. Run
2. Change one parameter
3. Re-run
4. Compare

You’ll quickly learn:
* What affects performance
* What affects accuracy
* Where compute is wasted

---

**What to Expect (Free Tier Limits)**

* Slower runs (shared system)
* Queue wait times
* Limited parallel jobs

--> This is normal and intentional

---

**When to Upgrade**

Upgrade makes sense if:
* You want faster runs
* You need multiple simulations at once
* You’re doing serious optimization

---

**Key Insight**

SimHPC is not just about running simulations.

It’s about:
> **Running → Changing → Comparing → Improving**

That loop is the core workflow.

---

**TL;DR (If You Skip Everything Else)**

Run a simulation → change one thing → run again → compare.

That’s how you unlock value.
